# batman.lego.game — affiliate bio page
`index.html` = the LEGO Batman bio-link page (game + sets, Amazon Associate tag `legobatman-20`).
## Deploy (1 step, when back at Mac)
1. `gh auth login`  (GitHub.com → HTTPS → login with browser)
2. `./deploy.sh`  → prints your live URL → put it in IG/TikTok bio.
## Before going live
- Confirm your Amazon tag is `legobatman-20` (Associates Central → Manage Tracking IDs). If different, find/replace it in index.html.
